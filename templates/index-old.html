Welcome to the home page

What is this?

This is a web app that I built for practice. It is a simple app that allows you to create a new user, add a new videogame, and view all of the videogames from the rawg API, plus those that you have added. It was built using Django, Python, and CSS.

How do I use it?

To create a new user, just click on the "New User" button at the top of the page. Once you have created a user, you can add a new videogame by clicking on the "Add Videogame" button. Once you have added a videogame, you can view all of the videogames that you have added by clicking on the "View Videogames" button.
